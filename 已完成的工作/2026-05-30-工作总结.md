## 一、MPD-PPO改进分析

### 1.1 核心问题诊断

200轮对比实验结果中，MPD-PPO(748.02) ≈ Vanilla PPO(748.23)，几乎无差异。深度分析发现：

| 问题 | 根因 |
|------|------|
| MPD-PPO前期收敛慢 | 固定紧裁剪(ε=0.1/0.2)限制了早期探索 |
| 差异化裁剪无意义 | ALF/OUT经tanh归一化后概率空间对称，不同ε不产生实质性差异 |
| 平滑正则冲突 | 固定SMOOTH_REG_WEIGHT=0.05在前期与reward信号冲突 |
| 后期偶发平滑崩溃 | Vanilla PPO在epoch 150+出现-36的平滑尖峰 |

### 1.2 MPD-PPO的真正优势

虽然最佳reward与Vanilla PPO持平，但MPD-PPO在**后期稳定性**上有显著优势：P_smooth在epoch 100+保持≈0，而Vanilla PPO在epoch 170+频繁出现-10到-36的平滑崩溃。轨迹平滑正则化是MPD-PPO最有价值的改进。

## 二、TAA-PPO：时间自适应退火PPO

### 2.1 设计理念

将MPD-PPO中无意义的"按动作类型(ALF/OUT)差异化裁剪"替换为有意义的"按时间维度差异化裁剪"：近端天(实际执行)紧裁剪，远端天(计划意图)松裁剪。

### 2.2 5项改进维度

| # | 改进 | 方案 | 验证结果 |
|---|------|------|:---:|
| 1 | **时间自适应裁剪** | 14天×2动作独立ε：day0-2(0.10)→day3-6(0.15)→day7-10(0.20)→day11-13(0.25) | ✅ 有效 |
| 2 | **裁剪预热调度** | 前80轮ε×1.8，线性退火到目标值 | ✅ 有效 |
| 3 | **自适应平滑正则** | 权重0.005(epoch<30)→线性增长→0.05(epoch≥100) | ✅ 有效 |
| 4 | **双Critic集成** | min(V1,V2)保守GAE | ❌ 训练不稳定 |
| 5 | **LayerNorm替代BN** | 小batch下更稳定 | ❌ 训练不稳定 |

改进4和5在当前小规模环境下导致训练不稳定，但理论上在大规模场景中有价值。最终TAA-PPO集成了改进1-3。

### 2.3 实现架构

```
TAA-PPO (algorithms/taa_ppo.py, ~250行)
├── Actor (复用ppo.Actor, BatchNorm)
├── Critic (复用ppo.Critic, BatchNorm)  
├── 时间自适应裁剪: base_eps[28] × warmup_factor
├── 自适应平滑正则: smooth_weight(epoch)
└── 标准PPO update循环
    ├── GAE (标准实现)
    ├── ratio[28] + per-index clipped_ratio[28]
    ├── actor_loss = -min(ratio×adv, clipped×adv).mean() + smooth_reg
    └── critic_loss = MSE(V(s), target)
```

### 2.4 参数配置

```python
# 时间自适应ε调度（14天，每行[ALF_eps, OUT_eps]）
TAA_EPS_SCHEDULE = [
    [0.10,0.10]*3, [0.15,0.15]*4, [0.20,0.20]*4, [0.25,0.25]*3
]
TAA_CLIP_WARMUP_EPOCHS = 80       # 预热80轮
TAA_CLIP_WARMUP_FACTOR = 1.8      # epoch0: ε×1.8
TAA_SMOOTH_START_WEIGHT = 0.005   # 初始平滑权重
TAA_SMOOTH_END_WEIGHT = 0.05      # 最终平滑权重
TAA_SMOOTH_RAMP_START = 30
TAA_SMOOTH_RAMP_END = 100
```

## 六、关键经验总结

1. **时间维度是比动作类型更有意义的差异化维度**：ALF/OUT在归一化空间对称，但day-1（执行）vs day-14（计划）有本质差异
2. **自适应调度比固定参数更有效**：裁剪预热和自适应平滑权重都证明了这一点
3. **Agent Loss的不同定义需要在可视化中处理**：PPO的loss（概率比，~0）和DDPG/SAC的loss（Q值，~1000）量级差3-4个数量级，必须用双Y轴
4. **LayerNorm和双Critic在小规模环境中不稳定**：不否定其理论价值，但需更大规模验证
5. **简化的loss计算带来意外性能收益**：TAA-PPO的28维统一裁剪比MPD-PPO的ALF/OUT分叉少一半操作，训练速度翻倍
6. **偏离基线图是展示微小差异的最佳方式**：当多个算法收敛到几乎相同的值，差值图比绝对值图有效得多
